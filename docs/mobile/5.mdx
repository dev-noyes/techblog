## 글로벌 환율 소개(Introduction)

👋 실시간으로 변하는 환율을 확인하고 싶으세요? 

## 사용가능한 OS

- 🍏 Android
- 🍎 iOS

### 주요 기능(Main Feactures)

글로벌 환율은 실시간 환율 뿐 아니라 히스토리 기능까지 제공합니다.

- 🍋 실시간 환율
- 🍌 과거의 기록까지
- 🍐 5분 단위의 기록까지 제공
- 🍊 역환율까지 제공
- 🍇 필터링 기능으로 원하는 환율만 제공
- 🍓 기준환율 변경
