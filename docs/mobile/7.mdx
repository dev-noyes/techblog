## 센서 고장 알리미 (Introduction)

👋 핸드폰에 들어있는 센서의 고장을 확인할 수 있는 서비스.

## 사용가능한 OS

- 🍏 Android
- 🍎 iOS

### 주요 기능(Main Feactures)

센서 고장 알리미 는 여러 센서들의 동작을 확인할 수 있습니다.

- 🍋 마이크
- 🍌 오디오
- 🍐 전면/후면 카메라
- 🍊 걸음센서
- 🍉 FACE ID 
- 🍎 TOUCH ID
